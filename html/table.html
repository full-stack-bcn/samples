A table showing the main HTML tags
Element Description
Structural Elements
<html> Represents the root (top-level element) of an HTML document, so it is also referred to as the root element.
All other elements must be descendants of this element.
<head> Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
<body> Represents the content of an HTML document.
There can be only one <body> element in a document.
Content sectioning
<main> Dominant content of the <body> of a document.
<article> Self-contained composition in a document, page, application, or site.
<section> A standalone section — which doesn't have a more specific semantic element to represent it — contained within an HTML document.
<header> Introductory content, typically a group of introductory or navigational aids.
<footer> Footer for its nearest sectioning content or sectioning root element.
<h1> – <h6> Headings of different levels.
<div> A generic marker for a section within a document.
Text-level tags
<p> A paragraph.
<br> A line break (empty tag).
<em> Emphasized text.
<strong> Strong text.
<code> Text that looks like computer code
<pre> A pre-formatted text (appears verbatim, including space).
<a> A hypertext link to another page (or another place in the current page).
<span> A generic marker for a piece of text within a paragraph.
Lists
<ol> An ordered list (shown with numbers).
<ul> An unordered list (shown with bullets).
<li> An list item.
Tables
<table> A table.
<tr> A table row.
<td> An table data cell.
<th> An table header cell.
<thead> A group of rows representing the heading of the table.
<tbody> A group of rows representing the body of the table.
<tfoot> A group of rows representing the footer of the table.
<caption> A description of the table (shown at the top).
